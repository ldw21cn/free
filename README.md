# free

At last, an osx substitution for linux's **free** command.

## Install

With [fisherman]

```
fisher free
```

## Usage

```fish
free
                 total     used     free   appmem    wired   compressed

+/- Cache:               1.24Gb   1.87Gb
Swap(43%):      2048Mb    877Mb   1171Mb
Load Avg:        1.63 1.95 1.95
```
